# BrickLink Skill

This repository contains the **Moltbot/Clawd skill** `bricklink` (BrickLink Store API / OAuth 1.0).

- For the full local usage guide (commands, auth setup, examples), see **[SKILL.md](./SKILL.md)**.
- CLI entrypoint: `scripts/bricklink.py`

The skill covers orders, store inventory (incl. mutating endpoints guarded by `--yes`), catalog, categories, colors, feedback, and notifications.
